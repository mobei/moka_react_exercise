##项目运行

# 需要有node环境，拿到项目（这里不放git地址了）
# 安装webpack(webpack2): npm install -g webpack
# 安装依赖: npm install 
# 启动服务： npm run dev 
# 打开浏览器：http://localhost:3000/   (默认端口3000,可以在config里面修改)

# 构建项目: npm run clean ， npm run build
# 构建生产环境： npm run clean ， npm run build-prod

# 构建后的项目在dist目录下
